- Run file index.html to use Compression Algorithm Examples.
- Input Area:
	+ Input text, maybe original text or compressed text.
- Output Area:
	+ Return the result.
- Compression Algorithm:
	+ RLC - Run Length Coding.
	+ LZW
	+ Huffman Coding
- Two mode: Compress or Decompress.
- Click Run button to run the app.
- Browse button to upload file.
	+ Cancel Read button to abort upload file.
- Download button to download output area content to file.
